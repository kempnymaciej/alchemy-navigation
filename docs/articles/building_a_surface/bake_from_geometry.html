Bake from geometry

The BakeFromMesh is a high level tool that allows you to create face holders based on the scene geometry. To use it, click Window->AlchemyNavigation->BakeFromMesh.

Window Property Description
Connection Radius (Settings) The maximum distance between vertices to be counted as one vertex.
Upwards (Settings) The upwards direction.
Max Slope (Settings) Maximum angle between Upwards and the normal of face.
Result Holder The parent object for the result face holders.
Result Position Specifies the way the results are placed.
Mesh Filters An array of the geometry sources.

As the results are baked you can modify them with standard features of the FaceHolder component.
