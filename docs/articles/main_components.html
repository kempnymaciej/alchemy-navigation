Main components

The AlchemySavigation provides many useful components that make work quick and convenient. Here is a list of the most important of them:

  1. Alchemy Navigation System - a component that provides key features for the entire system to work.
  2. Faces Holder - a high level component that allows you to create predefined elements of navigation mesh, edit them and move them around the scene in both edit and play mode.
  3. Simple Agent - allows you to create characters that intelligently move along navigation meshes to their destinations.
  4. Movement Modifier - allows you to create custom behaviours (such as obstacle avoidance) for your AI characters.
